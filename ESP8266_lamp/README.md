# How To Use

- 下载代码，打开本目录

- 修改global.cpp中的配置项

- 配置点片机，接线看下边的图

- 用arduino打开ESP8266_Clock_12864SSD1306.ino文件，选择单片机/串口，然后点击上传

- 刷好了之后，打开手机wifi，连上esp-8266命名的wifi，密码esp-8266

- 连上热点之后，打开浏览器，打开到192.168.4.1，然后配置WiFi，输入巴法云的密钥和主题名，点击提交
